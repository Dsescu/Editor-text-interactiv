# Editor Interactiv de Text

Aplicatie pentru crearea si editarea colaborativa a documentelor text.

## Functionalitati planificate

- **CRUD pentru fisiere**  
  Crearea, editarea, salvarea si stergerea fisierelor text, direct din interfata aplicatiei.

- **Formatarea textului**  
  Posibilitatea de a modifica fontul, dimensiunea, culoarea, alinierea textului si aplicarea diferitelor stiluri.

- **Editare colaborativa in timp real**  
  Mai multi utilizatori pot edita acelasi fisier simultan, de pe dispozitive diferite.  
  Sectiunile editate de fiecare utilizator vor fi evidentiate cu o **culoare distincta**, pentru a urmari contributiile.

- **Export si distribuire a documentelor**  
  Generarea documentelor in format unic si distribuirea acestora prin email.

## Tehnologii
- **Frontend:** React  
- **Backend:** Django (Python)
